Task 1.2 - Credit
Complete Task 1.1 and build your own custom React + Electron application:
Design and develop a simple but original React UI (e.g. a to-do app, calculator, note-taking tool,
or weather dashboard)
Ensure the app uses at least one React component with state management (useState or
useReducer)
Package the app as a distributable installer using electron-builder
Include an app icon and a custom application name in the build configuration
Demonstrate the installed app running on your machine with screenshots

Task 1.3 - Distinction
Complete Task 1.2 and publish your application as a GitHub Release:
Refactor your app to use multiple React components and at least one external npm dependency
(e.g. axios, chart.js, date-fns)
Configure electron-builder to produce release artifacts (installer + unpacked build)
Create a GitHub repository for your project and tag a versioned release (e.g. v1.0.0)
Upload the packaged installer(s) as assets to the GitHub Release
Provide a link to the public GitHub Release and screenshots of the release page

Task 1.4 - High Distinction
Complete Task 1.3 and implement in-app auto-update functionality using electron-updater:
Integrate the electron-updater package into your Electron main process
Configure electron-builder to publish releases to GitHub (provider: github in electron-builder
config)
Implement update checking on app start - notify the user when an update is available
Release a v1.0.0 and a v1.1.0 (with a visible change) to GitHub Releases
Demonstrate the auto-update flow: app running on v1.0.0 detects and installs v1.1.0
Provide screenshots or a screen recording showing the full update lifecycle
